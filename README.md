# Ember-ufo-loading
I just wrapped this amazing loading animation from [CodePen](http://codepen.io/biacosta/pen/QwBdBB) by [biacosta](http://codepen.io/biacosta/)

## Installation

* `ember install ember-ufo-loading` this repository

## Using

```hbs
{{ufo-loading}}
```
